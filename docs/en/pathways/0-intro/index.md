# About

Welcome to the SAS Viya Forge Pathways. Pathways provide a curated list of documents that guide you on the road from designing, through to delivering and eventually operating and maintaining your environment. The documents listed in each pathway can also be read as stand-alone documents, but using them in combination with the other documents in a pathway provides the best overview of a particular topic.

You can choose a relevant pathway in the navigation pane on the right of this page.

!!! note

    Can't find what you need? You can contribute to Viya Forge using the instructions on our [Contribute](../../contributing.md) page.
    You can also request a new document by creating a [new Issue](https://github.com/sassoftware/sas-viya-forge/issues/new/choose) in our GitHub project or vote for an [existing request](https://github.com/sassoftware/sas-viya-forge/issues).
