# About

SAS maintains a repository of tuning parameters designed to improve overall performance for large scale Viya deployments.

This project is broken down into a set of configuration "tunables" for various scenarios SAS Viya users perform on a routine basis. Each scenario represents an action, or a set of actions, users may perform while interacting with SAS Viya web applications, command-line interfaces, scheduled batch jobs, mobile applications, etc.

## When to use

Each scenario describes the situation in which it is applicable. Some scenarios like the platform-common scenario will be applicable for environments with a large number of users. Other scenarios like the scoring-execution scenario will be applicable for environments with high volumes of requests to the Micro Analytic Server. This will differ from scenario to scenario.
