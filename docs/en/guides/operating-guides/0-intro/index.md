# About

Welcome to the SAS Viya Forge Operating Guides. Operating guides provide practical guidance on operating and maintaining your SAS Viya environments. The procedures described in operating guides are usually executed while the environment is already being used by end-users and serve to maintain the functionality and value a SAS Viya environment provides. Operating Guides can evolve over time as the Viya platform or it's ecosystem changes.

You can choose a relevant operating guide in the navigation pane on the right of this page.
